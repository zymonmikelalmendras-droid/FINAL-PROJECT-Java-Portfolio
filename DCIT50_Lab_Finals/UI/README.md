Contains classes for all windows and forms the user interacts with.

MainWindow.java
 - The main window of the application. Displays tasks in a table and includes the “Add Task” button. Refreshes the table whenever a new task is added.

TaskForm.java
 - A separate form for creating new tasks. Contains fields for ID, name, description, and status dropdown. Validates input, sends the task to TaskManager, and closes after submission.
